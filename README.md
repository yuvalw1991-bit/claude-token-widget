# Claude Context Window Widget

ווידג'ט צף לWindows שמציג בזמן אמת את ניצול חלון ההקשר של Claude.

---

## מה זה עושה?

- **מונה טוקנים חי** — מתעדכן כל 0.8 שניות מקבצי הסשן המקומיים של Claude
- **סרגל צבעוני** — ירוק → כתום → אדום ככל שחלון ההקשר מתמלא
- **עוקב אחרי Claude** — נצמד לפינה הימנית-תחתונה של Claude Desktop, Antigravity IDE, או VS Code / Cursor עם Claude Code
- **מצב מינימלי** — כיווץ לפיל קטן שמציג רק את האחוז
- **גרירה חופשית** — ניתן להזיז לכל מקום על המסך
- **אפס ענן, אפס טלמטריה** — קורא רק קבצים מקומיים מתוך `~/.claude/projects/`

## התקנה והפעלה

1. התקיני **AutoHotkey v2** ו-**Node.js** אם עדיין לא מותקנים
2. הורידי את הקבצים (או שכפלי את ה-repo)
3. לחצי דאבל-קליק על **`הפעלה.bat`**

הווידג'ט יופיע בפינה הימנית-תחתונה של חלון Claude.

> **הפעלה אוטומטית עם Windows:** לחצי דאבל-קליק על `התקן_אוטוסטרט.bat` — יוצר קיצור דרך בתיקיית Startup אוטומטית.

## כפתורי הווידג'ט

| כפתור | פעולה |
|-------|-------|
| `R` | רענן — קריאה מחדש מיידית של קובץ הסשן הפעיל |
| `-` | כווץ — מעבר למצב פיל קטן |
| `x` | סגור — הסתרת הווידג'ט (ממשיך לרוץ ברקע) |

**תפריט מגש המערכת** (לחיצה ימנית על אייקון AHK ליד השעון):
`Show / Hide` · `Refresh` · `Reset` · `Exit`

**להחזרת הווידג'ט לאחר סגירה:** לחיצה ימנית על ה-tray ← Show / Hide

## איך זה עובד?

`watcher.js` (Node.js) סורק את `~/.claude/projects/` לאיתור קובץ הסשן האחרון שנשמר (`.jsonl`), קורא את הכמה KB האחרונים, ומחלץ את שימוש הטוקנים מהשדה `usage` ביומן השיחה. התוצאה נכתבת לקובץ זמני (`%TEMP%\claude_tokens.json`).

`claude_token_widget.ahk` (AutoHotkey v2) מסקר את הקובץ הזמני כל 0.9 שניות ומרנדר את הממשק.

**חישוב:** `input_tokens` + `cache_creation_input_tokens` + `cache_read_input_tokens`
**מקסימום:** 200,000 טוקנים (Claude 3.x / 4.x)

---

## אבטחה

- `watcher.js` מאמת נתיבים עם `fs.realpathSync` — מונע path traversal וסymlink escape
- קבצים ותיקיות שהם symlinks מדולגים בסריקה
- קבצים גדולים מ-50MB מדולגים
- אין בקשות רשת, אין תלויות חיצוניות
