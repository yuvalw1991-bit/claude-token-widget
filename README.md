# Claude Context Window Widget

> ווידג'ט צף לWindows שמציג כמה מקום נותר בחלון ההקשר של Claude — בזמן אמת.

![preview](https://img.shields.io/badge/Windows-10%2F11-blue) ![ahk](https://img.shields.io/badge/AutoHotkey-v2-green) ![node](https://img.shields.io/badge/Node.js-18%2B-brightgreen)

---

## ⬇️ הורדה והתקנה — 3 שלבים

### שלב 1 — הורידו את הקובץ

**[📦 לחצו כאן להורדה](https://github.com/yuvalw1991-bit/claude-token-widget/releases/latest/download/claude_token_widget_v1.0.zip)**

חלצו את קובץ ה-ZIP לתיקייה כלשהי במחשב.

---

### שלב 2 — התקינו את הדרישות (פעם אחת בלבד)

| תוכנה | למה צריך אותה | קישור הורדה |
|-------|--------------|-------------|
| **AutoHotkey v2** | מריץ את הווידג'ט | [הורד מ-autohotkey.com](https://www.autohotkey.com/) |
| **Node.js** | קורא את נתוני הטוקנים | [הורד מ-nodejs.org](https://nodejs.org/) |

> אם כבר מותקנים — דלגו ישירות לשלב 3.

---

### שלב 3 — הפעילו

לחצו דאבל-קליק על **`הפעלה.bat`**

זהו! הווידג'ט יופיע בפינה הימנית-תחתונה של חלון Claude.

---

### 🔁 הפעלה אוטומטית עם Windows (אופציונלי)

לחצו דאבל-קליק על **`התקן_אוטוסטרט.bat`** — הווידג'ט יעלה לבד עם כל הפעלת Windows.

---

## 📺 מה רואים?

הווידג'ט נצמד לפינת חלון Claude ומציג:
- **אחוז השימוש** בחלון ההקשר
- **מספר הטוקנים** מתוך המקסימום
- **סרגל צבעוני** שמשתנה לפי המצב:

| צבע | משמעות |
|-----|--------|
| 🟢 ירוק (0–59%) | הרבה מקום נותר |
| 🟠 כתום (60–81%) | חצי בשימוש |
| 🔴 אדום (82–100%) | קרוב לגבול — כדאי לפתוח שיחה חדשה |

---

## 🖱️ כפתורי הווידג'ט

| כפתור | פעולה |
|-------|-------|
| `R` | רענן ידני — שימושי בעת מעבר בין שיחות |
| `-` | כווץ למצב מינימלי |
| `x` | הסתר (לא סוגר — ממשיך לרוץ ברקע) |

**להחזרת הווידג'ט:** לחיצה ימנית על האייקון ליד השעון ← **Show / Hide**
