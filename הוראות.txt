Claude Context Window Widget — הוראות
======================================

מה זה?
-------
ווידג'ט קטן שמציג בזמן אמת את אחוז ניצול חלון ההקשר (context window)
של Claude. מופיע בפינה ימנית-תחתונה של חלון Claude, מעל כל חלון אחר.
תומך ב: Claude Desktop, Antigravity IDE, VS Code / Cursor עם Claude Code.


דרישות
------
  • Windows 10 / 11
  • AutoHotkey v2  ->  https://www.autohotkey.com/
  • Node.js 18+    ->  https://nodejs.org/


הפעלה
------
לחיצה כפולה על:  הפעלה.bat

זה כל מה שצריך. הווידג'ט ה-watcher עולים יחד.


כפתורי הווידג'ט
----------------
  R   רענן  — קריאה מחדש של קובץ הסשן הפעיל
  -   כווץ  — מצב מינימלי: רק האחוז + כפתורים
  x   סגור  — הסתרה (הווידג'ט ממשיך לרוץ ברקע)

תפריט מגש המערכת (לחיצה ימנית על אייקון AHK בשורת המשימות):
  Show / Hide  |  Refresh  |  Reset  |  Exit

גרירה: לחץ וגרור את הווידג'ט לכל מקום על המסך.
להחזרת הווידג'ט לאחר סגירה: לחיצה ימנית על ה-tray -> Show / Hide


עדכונים אוטומטיים
------------------
  • בתוך שיחה פעילה — מתעדכן כל 0.8 שניות
  • מעבר לשיחה אחרת — לחץ R לעדכון מיידי


צבעי סרגל ההתקדמות
-------------------
  ירוק   0-59%    — שפע מקום
  כתום  60-81%    — מחצית בשימוש
  אדום  82-100%   — קרוב לגבול — שקול לפתוח שיחה חדשה


חישוב האחוז
-----------
  סה"כ = input_tokens + cache_creation_tokens + cache_read_tokens
  מקסימום = 200,000 טוקנים (Claude 3.x / 4.x)


הפעלה אוטומטית עם Windows
---------------------------
לחיצה כפולה על:  התקן_אוטוסטרט.bat
(יוצר קיצור דרך בתיקיית Startup של Windows)
